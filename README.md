# React Music Player

Music player project built with React.

## Tech Stack

- React
- Create React App

## Getting Started

```bash
npm install
npm start
```

## Available Scripts

- `npm start` - run development server
- `npm test` - run tests
- `npm run build` - production build
- `npm run eject` - eject CRA configuration
